## 📖 项目简介

**aegis 系统** 是一款基于角色的访问控制（RBAC）权限管理系统，采用 **Spring Boot + Vue** 前后端分离架构。

- 前端：Vue、ElementUI、Vuex、Vue-Router、Axios
- 后端：Spring Boot、MyBatis-Plus、Spring Security + JWT、Redis
- 功能特性：动态菜单加载、URL 级权限控制、按钮级权限控制
